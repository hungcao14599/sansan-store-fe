# Sansan POS Frontend

Frontend POS dùng React + Vite + Zustand + React Query + TailwindCSS với UI custom.

## Lệnh chạy

```bash
npm install
cp .env.example .env
npm run dev
```

## Màn hình

- Dashboard
- POS bán hàng với thuế tự động theo sản phẩm và payment method
- Quản lý sản phẩm
- Bảng giá
- Hóa đơn
